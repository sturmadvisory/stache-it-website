## Apple Search Ads — Getting Started

### Setup
1. Go to https://searchads.apple.com
2. Sign in with your Apple Developer account
3. Create a campaign for Stache-it (App ID: 6748416239)

### Recommended keywords (based on your ASO)
**High intent (exact match):**
- siri notes
- voice memo app
- quick notes app
- memory vault
- fact saver
- save notes siri

**Broader (broad match):**
- remember app
- personal notes
- secure notes iphone
- icloud notes alternative
- brain dump app

### Budget suggestion
- Start small: $5-10/day
- Set a max CPA (cost per acquisition) of $1-2
- Run for 2 weeks, then evaluate which keywords convert
- Pause keywords with high spend but no installs

### Tips
- Apple Search Ads Basic is simpler (set budget, Apple picks keywords)
- Search Ads Advanced gives you keyword control (recommended)
- Your ASO keywords and App Store description feed into Basic's algorithm
